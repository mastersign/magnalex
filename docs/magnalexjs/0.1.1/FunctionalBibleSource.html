Class: FunctionalBibleSource

FunctionalBibleSource(id, name, hasTranslation, hasBook, loadBook)

An implementation of BibleSource using three functions to do the actual work.

Comes with an integrated cache for the loaded books, so that no book has to be loaded twice.
